# 고성훈 (koesunghoon) 👋

**MCU 기반 제어 시스템을 구현하는 임베디드 엔지니어**

- RTOS 기반 실시간 제어 시스템 개발 경험
- UART · SPI · I2C · TCP/IP 통신 구현 경험
- HW-SW 통합 설계 및 디버깅 경험
